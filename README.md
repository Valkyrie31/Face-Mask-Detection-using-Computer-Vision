# Face-Mask-Detection-using-Computer-Vision
# 😷 Face Mask Detection using Computer Vision

## 📌 Overview

This project detects whether a person is wearing a face mask or not using Computer Vision and Machine Learning techniques. It uses image classification to identify two classes: **Mask** and **No Mask**.

---

## 🎯 Features

* Detects faces in real-time
* Classifies mask vs no mask
* Uses pre-trained models for better accuracy
* Simple and fast implementation

---

## 🛠️ Technologies Used

* Python
* OpenCV
* TensorFlow / Keras
* NumPy

---

## 📂 Project Structure

* dataset/ → contains images (mask & no mask)
* train.py → model training file
* detect.py → real-time detection
* model.h5 → trained model
* README.md → project documentation

---

## ▶️ How to Run

1. Install dependencies:
   pip install opencv-python tensorflow numpy

2. Train the model:
   python train.py

3. Run detection:
   python detect.py

---

## 📊 Output

* Displays webcam feed
* Shows label: "Mask" or "No Mask"
